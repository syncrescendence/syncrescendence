# Oracle Anti-Patterns

## Core Failures

### 1. Stateless prompting

Failure mode:
- generic answers
- no repo specificity
- low signal

### 2. Weak content proof

Failure mode:
- paraphrased “quotes”
- fabricated file engagement
- unjustified confidence

### 3. Treating synthesis as verification

Failure mode:
- product reality errors
- invalid ratification
- architecture built on stale capability assumptions

### 4. Rewarding rhetorical certainty

Failure mode:
- inflated ecosystem claims
- false closure
- loss of good partial signal hidden inside overreach
