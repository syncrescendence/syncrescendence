# Perplexity Anti-Patterns

## Core Failures

### 1. Architecture outsourcing

Failure mode:
- shallow design
- overfitting to current docs
- weak institutional fit

### 2. Citation worship

Failure mode:
- assuming cited means correct
- ignoring source quality
- importing weak secondary evidence

### 3. Vague verification requests

Failure mode:
- diffuse answers
- poor evidentiary focus
- weak usefulness for actual design decisions

### 4. Role collapse with Oracle

Failure mode:
- no clean triangulation
- duplicated surface value
- loss of synthesis vs verification distinction
